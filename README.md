🌸 MOOD-VS-PRODUCTIVITY-AI-ANALYZER

💡 About the Project

This project is a simple and practical attempt to understand how our mood, study time, and sleep affect our daily productivity.

As students, we often feel that some days are super productive while others are not — this project helps to analyze that pattern using Python, graphs, and basic machine learning.

🎯 Objective

The main goal of this project is to:

- Track daily habits (mood, study, sleep)
- Calculate a productivity score
- Visualize patterns using graphs
- Predict productivity using a simple ML model

⚙️ How It Works

1. The user enters:
   
   - Mood (happy / neutral / sad)
   - Study hours
   - Sleep hours

2. The program:
   
   - Converts mood into a numeric value
   - Calculates productivity using a formula
   - Stores the data for analysis

3. After multiple entries:
   
   - Graphs are generated
   - A Machine Learning model predicts productivity
   - Average productivity is calculated

🧠 Logic Behind Productivity

The productivity score is calculated using:

Productivity = (Mood × 2 + Study × 1.5 + Sleep) / 3

This means:

- Mood has a strong impact
- Study hours contribute significantly
- Sleep supports overall performance

📊 Features

- ✔️ Simple user input system
- ✔️ Productivity calculation
- ✔️ Smart suggestions based on inputs
- ✔️ Graph visualization (Matplotlib)
- ✔️ Machine Learning prediction (Linear Regression)
- ✔️ Average productivity tracking

📈 Graphs Used

- Mood vs Productivity
- Study Hours vs Productivity

These graphs help in understanding trends visually.

🤖 Machine Learning

The project uses Linear Regression to:

- Learn from user data
- Predict future productivity

Note: At least 3 entries are required for prediction.

💬 Suggestions System

The program also gives helpful suggestions like:

- Improve study time
- Get enough sleep
- Take breaks if mood is low
